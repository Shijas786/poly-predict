✅ Vercel is perfect for 24/7 deployment — this is a pure static HTML app. No backend, no database, no server costs. All data fetched client-side from public APIs.
1

Initialize Git Repo

In your terminal, run:

mkdir polymarket-pulse && cd polymarket-pulse
git init
cp /path/to/pp-intelligence.html index.html
git add . && git commit -m "Initial deploy"
2

Push to GitHub

Create a new repo at github.com/new, then:

git remote add origin https://github.com/YOUR_USERNAME/polymarket-pulse.git
git branch -M main
git push -u origin main
3

Deploy to Vercel (One Command)

Install Vercel CLI once: npm i -g vercel
Then deploy:

vercel --prod

Your dashboard is live at https://polymarket-pulse.vercel.app in ~30 seconds. Auto-updates on every git push.

4

Auto-refresh Every 5 Min

Built in — the dashboard re-runs the full backtest every 5 minutes automatically. Live win rates update in real-time as new rounds complete.

5

APIs Used (All Free, No Auth)

gamma-api.polymarket.com/events  → round outcomes
data-api.polymarket.com/trades   → wallet trade data
clob.polymarket.com/book         → order book depth
api.binance.com/api/v3/klines    → BTC candle data
