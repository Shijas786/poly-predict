# Polymarket Pulse Intelligence Dashboard

Real-time BTC 5-min strategy intelligence dashboard.

## Features
- 55 on-chain trading strategies with live win rates
- Enhanced calendar with strategy overlay per day
- Wallet profiling (oracle, manipulator, cluster detection)
- Price bucket analysis (50-60¢ to 90-99¢)

## Deploy
```bash
npm i -g vercel
vercel --prod
```

## APIs Used (all free, no auth)
- gamma-api.polymarket.com
- data-api.polymarket.com
- clob.polymarket.com
- api.binance.com
